## 🛠 Cara Menjalankan Proyek

1. Clone repositori:
   ```bash
   git clone https://github.com/bifrace/farrelrafael-web-projects.git
   cd farrelrafael-web-projects


### 📦 Masuk ke folder proyek yang ingin dijalankan:
```bash
cd nama-proyek
```
### Install dependensi:

```bash
npm install
```
### Jalankan proyek:

```bash
npm run dev
```
